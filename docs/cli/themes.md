# 主題

Gemini CLI 支援多種主題，可自訂其配色方案和外觀。您可以透過 `/theme` 指令或 `"theme":` 設定來變更主題以符合您的偏好。

## 可用主題

Gemini CLI 提供多種預先定義的主題，您可以在 Gemini CLI 中使用 `/theme` 命令列出這些主題：

- **深色主題**：
  - `ANSI`
  - `Atom One`
  - `Ayu`
  - `Default`
  - `Dracula`
  - `GitHub`
- **淺色主題**：
  - `ANSI Light`
  - `Ayu Light`
  - `Default Light`
  - `GitHub Light`
  - `Google Code`
  - `Xcode`

### 變更主題

1.  在 Gemini CLI 中輸入 `/theme`。
2.  出現對話方塊或選取提示，列出可用的主題。
3.  使用方向鍵選取主題。某些介面可能會在您選取時提供即時預覽或反白顯示。
4.  確認您的選取以套用主題。

### 主題持續性

選取的主題會儲存在 Gemini CLI 的[設定](./configuration.md)中，因此您的偏好設定會在工作階段之間保留。
